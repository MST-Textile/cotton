Usage

Cotton.js can be invoked by initializing its init method. The method should be used to set of links, that will act as the toolbar formatting buttons. The actions that these buttons do when clicked, are configured by defining data attributes in the tags themselves. This allows you to configure any set of formatting actions, or pick only the ones you need for a particular field, without having to write different JavaScript implementations. For instance a field that only allows single line of content, won't need any block tag options. The plugin can be included and invoked by using either browser globals or AMD.

<script src="jquery.js"></script>
<script src="cotton.js"></script>
<script>
    $('.cotton a').cotton();
</script>

The above will include jQuery library and then the plugin source from cotton.js file. The plugin is initialized and set to use .cotton a elements as its formatting buttons. Set of example buttons could look something like this:

<p class="cotton">
    <a href="#body" data-cotton-callback="inline" data-cotton-before="*" data-cotton-after="*" data-cotton-shortcut="b"><b>B</b></a>
    <a href="#body" data-cotton-callback="inline" data-cotton-before="_" data-cotton-after="_" data-cotton-shortcut="i"><i>I</i></a>
    <a href="#body" data-cotton-callback="inline" data-cotton-before="+" data-cotton-after="+" data-cotton-shortcut="u"><ins>U</ins></a>
    <a href="#body" data-cotton-callback="inline" data-cotton-before="-" data-cotton-after="-" data-cotton-shortcut="o"><del>O</del></a>
    <a href="#body" data-cotton-callback="heading" data-cotton-tag="h2" data-cotton-shortcut="h">h#</a>
    <a href="#body" data-cotton-callback="link">@</a>
    <a href="#body" data-cotton-callback="list" data-cotton-bullet="*">*</a>
    <a href="#body" data-cotton-callback="list" data-cotton-bullet="#">#</a>
    <a href="#body" data-cotton-callback="code" data-cotton-before="@" data-cotton-after="@">{}</a>
    <a href="#body" data-cotton-callback="block" data-cotton-tag="bq">’</a>
    <a href="#body" data-cotton-callback="acronym">ABC</a>
    <a href="#body" data-cotton-callback="inline" data-cotton-before="!" data-cotton-after="!">img</a>
</p>

<p><textarea id="body" name="body" rows="4" cols="30"></textarea></p>

These buttons can be included anywhere in your document.

Supports

Web browsers

Cotton.js will run on any modern web browser. The base browser compatibility includes Internet Explorer 9 and 10, and the last two stable releases of Firefox, Chrome, Safari and Opera.

Internet Explorer 8 and older are not supported as Cotton.js uses no platform-specific abstraction and exceptions outside from the one offered by jQuery.

AMD

In addition to normal browser global use, Cotton.js can be used with RequireJS and other AMD loaders. The plugin registers itself as an anonymous module, if the use of an AMD API is detected.

require(['jquery', 'cotton'], function ($) {
    $('.cotton a').cotton();
});

With the default RequireJS configuration, the above would load two files, scripts/jquery.js and scripts/cotton.js, and then attach the toolbar functionality to .cotton a elements present on the document at the time.

jQuery

Currently Cotton.js supports jQuery library versions starting from 1.7. This includes versions such as, but not limited to, 1.7.2, 1.8.3, 1.9.1, 1.10.1 and 2.0.2. See tests directory for all versions the project is tested against.
